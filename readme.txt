这是一个基于人工智能（AI）的虚拟键盘项目。该项目具有以下特性：
特性：
实时追踪手势
可以根据手势在文本编辑器中实时输入文字

安装步骤：
安装所有依赖项：运行 libraries.bat 或者执行 pip install -r requirements.txt
运行程序：python main.py

帮助：
如果你遇到摄像头未显示或出现错误的问题，尝试更改以下行的数值（例如更改为1）：cap = cv2.VideoCapture(0)。递增这个数字直到你看到摄像头为止。

虚拟键盘：
“TRUN”中英文转换（需要将您的键盘的中英文转换改为Crtl）

点击：
为了模拟点击，你需要连接手上的食指和中指。
